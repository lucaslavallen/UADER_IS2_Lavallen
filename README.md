# Ingeniera en software

## Instalación y Uso

### Requisitos
- Node.js
 
### Instalación
```bash
# Clonar el repositorio
git clone https://github.com/usuario/venta-pulseras.git
cd venta-pulseras
```

### Ejecución
```bash
# Iniciar servidor
dnpm start
```
